Position Importance Formula

Variable Definitions:
$p$ = position (QB, WR, RB, TE)
$\alpha(p)$ = PAR value for position p
$\delta(p)$ = dropoff for position p
$\sigma(p)$ = scarcity ratio for position p
$\epsilon_{10}(p)$ = elite 10% level for position p
$\epsilon_{20}(p)$ = elite 20% level for position p
$\phi(p)$ = final importance score for position p

Dropoff Function

$\delta(p) = \epsilon_{10}(p) - \epsilon_{20}(p)$

Raw Importance Score

$\psi(p) = \frac{\alpha(p) + \delta(p)}{2\sigma(p)}$

Final Normalized Importance

$\phi(p) = \frac{\psi(p)}{\min_{p'} \psi(p')}$

Complete Formula (Expanded)

$\phi(p) = \frac{\frac{\alpha(p) + \epsilon_{10}(p) - \epsilon_{20}(p)}{2\sigma(p)}}{\min_{p'} \left(\frac{\alpha(p') + \epsilon_{10}(p') - \epsilon_{20}(p')}{2\sigma(p')}\right)}$
